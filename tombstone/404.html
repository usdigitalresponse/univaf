COVID-19 Unified Vaccine Appointment Finder API (UNIVAF)

This API is no longer maintained as of June 15, 2023.

The federal COVID-19 emergency has ended, and U.S. Digital Response is no longer providing active support or updates for the API and the code behind it.

This system is open-source software, and you can deploy your own version of it based on the code at github.com/usdigitalresponse/univaf. If you need additional information or support, please contact univaf@usdigitalresponse.org.

From March 2021 – June 2023, U.S. Digital Response (USDR) built and maintained a public API providing combined, standardized, live information about COVID-19 vaccine appointment availability at locations across the United States at https://getmyvax.org/. Data was sourced from a variety of official data sources (e.g. CVS Pharmacy) and unofficial data sources (e.g. web scraping and volunteer phone banking). It was built in partnership with the State of New Jersey Office of Innovation, the Municipality of Anchorage Innovation Team, VaccineSpotter.org, and Vaccinate the States and acted as a source of information for a variety of government and community vaccine finder websites.

  • 130 million Appointments Published
  • 44,059 Pharmacies and Clinics
  • 53 States and Territories Covered
  • 808 days In Operation

…and supported development of a federal data standard.

The live API has been shut down, but you can find the archived source code and documentation at github.com/usdigitalresponse/univaf. Historical data and logs are also available for analysis at archives.getmyvax.org. For current information about vaccine locations, please check vaccines.gov.

You can also fork and deploy your own copy of the system. Please keep in mind that some sources may have moved, changed formats, or stopped working by the time you do so — this system requires continual monitoring maintenance in order to keep up to date with changes to all the systems it interacts with. Some systems also require authorization via API keys or other tokens, which you will need to obtain separately.
